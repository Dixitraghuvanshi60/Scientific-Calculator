# Scientific Calculator

A Scientific Calculator built using Python and Tkinter that provides a simple graphical user interface for performing basic and scientific mathematical operations.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧮 Modulus
- 📐 Square Root
- 🔢 Power
- 📊 Trigonometric Functions (Sin, Cos, Tan)
- 📈 Logarithmic Functions
- 🖥️ User-friendly GUI using Tkinter

## Technologies Used

- Python 3
- Tkinter

## Project Structure

Scientific-Calculator/

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Dixitraghuvanshi60/Scientific-Calculator.git
```

2. Navigate to the project folder:

```bash
cd Scientific-Calculator
```

3. Run the application:

```bash
python Scientific_calculator.py
```

## Requirements

- Python 3.x
- Tkinter (included with most Python installations)
